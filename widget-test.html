How to Test:

  • Step 1: Start your Next.js dev server (npm run dev)
  • Step 2: Go to /dashboard/embed and copy the generated code
  • Step 3: Replace the script tag above with your generated code
  • Step 4: Open this HTML file in your browser
  • Step 5: Click the button to test the booking modal

Note:

For production, change http://localhost:3000 to your actual domain. The widget will automatically work on any website once deployed.
